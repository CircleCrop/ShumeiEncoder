# 树莓编码 / ShumeiEncoder

这是一个面向苏州中学树莓社团内的编码工具。

适用平台：`Windows 10/11 (x86 64bit)`，未来适配测试后，将在 `MacOS (ARM)` 上推出。

我们致力于创建面向创作者的软件。抛开繁冗的参数，你只需要明确格式需求――剩下的请交给工具。

## Support Formats
### Video
Decode:

	Apple Prores Series, Avid DNx Series, H.264/AVC, H.265/HEVC
	4:2:0/4:2:2/4:4:4 8/10bits

Encode:

	H.264/AVC, H.265/HEVC
	4:2:0 8/10bits

### Audio

	Decode：AAC, FLAC, WAV
	Encode：AAC, FLAC

### Container

	MP4, MOV, MKV, M4A (Audio Only), FLAC, WAV
