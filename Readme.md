# 树莓编码 / ShumeiEncoder

**树莓编码**是一款面向**苏州中学树莓社**的编码工具链。

支持平台：  
- **Windows 10/11 (x86 64bit)**  

---

## 🛠 核心功能

1. **高效编解码**：  
   - 实现对主流视频/音频编解码格式的兼容。
   - 支持 `4:2:0` / `4:2:2` / `4:4:4` 采样。
   - 最大化利用 CPU/GPU 编解码能力。

2. **用户友好性**：  
   - 屏蔽底层复杂参数，自动调整最佳性能配置。

---

## 📹 视频

### **解码 (Decode)**

- 格式：  
  - **Apple ProRes**
  - **Avid DNxHD / DNxHR**
  - **H.264/AVC**  
  - **H.265/HEVC**
- 色彩采样：4:2:0 / 4:2:2 / 4:4:4  
- 位深：8-bit / 10-bit  

### **编码 (Encode)**

- 格式：  
  - **H.264/AVC**  
  - **H.265/HEVC**
- 色彩采样：4:2:0  
- 位深：8-bit / 10-bit  

---

## 🎵 音频

### **编码 (Encode)**
